Lo que hago

Desarrollo software real en producción usando Angular, .NET y Electron, con enfoque en claridad, arquitectura limpia e integraciones fiables.

Aplicaciones Web (Angular) Componentes modulares, formularios avanzados, patrones de estado, i18n, UI limpia y accesible. Experiencia en paneles, datatables, dashboards e interfaces empresariales.
APIs y Backends con .NET / C# APIs REST con validación robusta, Entity Framework, seguridad básica, arquitectura por capas y servicios separados. Integración con bases de datos SQL reales.
Aplicaciones de Escritorio (Electron) Angular + Electron, integración con SO, control de multimedia, drag & drop, persistencia local y entregas listas para producción.
Integraciones y Servicios Externos MQTT con dispositivos (cámaras, barreras, sensores), Redsys (firma 3DES), Google APIs, SendGrid y APIs de terceros en entornos reales.
Bases de Datos & Consultas SQL Server, PL/SQL, modelos de datos, migraciones, consultas optimizadas y gestión de datos para web y escritorio.
DevOps & Flujo de Trabajo Docker, entornos de prueba, control de versiones, CI/CD básico, ramas limpias, PRs claros y colaboración con buenas prácticas.
